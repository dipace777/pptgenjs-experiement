# ppty

## Getting started (first-time setup)

### Prerequisites

- **Node.js** 20+ (the project uses Vite 8 and React 19)
- **pnpm** — install with `npm install -g pnpm` if you don't have it
- An **OpenAI API key**

### 1. Clone and enter the project

```bash
git clone <repo-url>
cd presenton/experiments/ppty
```

### 2. Install dependencies

```bash
pnpm install
```

### 3. Configure environment variables

Copy the example file and fill in your OpenAI key:

```bash
cp .env.example .env
```

Then edit `.env`:

```
OPENAI_API_KEY=sk-...
```

### 4. Start the dev server

```bash
pnpm dev
```

Vite will print a local URL (typically `http://localhost:5173`). Open it in your browser.

## Troubleshooting

- **"OPENAI_API_KEY is not defined"** — make sure `.env` exists in the project root and contains a valid key, then restart the dev server.
- **pnpm not found** — install it globally with `npm install -g pnpm` or via [the official installer](https://pnpm.io/installation).
- **Port 5173 in use** — Vite will pick the next free port automatically; check the terminal output for the actual URL.
